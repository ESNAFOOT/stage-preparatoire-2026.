2. Clique sur le crayon ✏️.
3. Cherche : programme-section
4. Modifie les dates, thèmes ou contenus dans les cartes.
5. Clique sur Commit changes.

Le programme est présenté en cartes modernes, plus lisible sur téléphone qu'un tableau.
